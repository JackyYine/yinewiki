AXON HYDRA RT

 

1 安装

  1. 去 C/Users/PointsMe 下面创建一个文件夹 名为 Axon
  2. 安装好AXON的驱动软件，驱动是 AXONFPID

设置IP

按 CHIAVE 到 PROGRAMMAZIONE

确认，然后移到 ETHERNET

SI / DHCP NO / PROTOCOL TCP / PROTOCOL IN ETH? SI

SET IP ADDRESS IP地址 / REMOTE IP 不设置 / COM. PORT 9101 / 

GATEWAY 网关地址 / PRIMARY DNS 8.8.8.8 / SECONDARY DNS 8.8.4.4

MASK 255.255.255.0 / 其他的都不设置

 

改IVA

 先做个CHIUSURA

在驱动里点 CONFIGURAZIONE STAMPANTE

点击LEGGI IVA 和 LEGGI REPARTI

改成这样（最主要的是 REPARTO 2 用IVA 10%）

点击 INVIA IVA

 

设置开机自启 win+R 然后 shell:startup
