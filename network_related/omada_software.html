Omada网络管理软件

Omada是用于统一管理最新一代TP-Link网络设备的软件。

常用的界面为以下几种:

Settings → Wired Networks → Wan 设置内网路由器在一级网络的IP

Settings → LAN 设置内部网络

Settings → Wireless Networks 设置内部网络的WIFI参数

Settings → Transmission → NAT 设置路由器的端口映射

Devices → 路由器图标 → WAN 在此可以查看当前路由器在一级网络的IP

Devices → 发射器图标 → Config → IP Settings 设置发射器IP

 

设备界面


第一个黑色图标为路由器，第二个与第三个图标为发射器图标:

  • 绿色的Connected为已连接
  • 蓝色的Pending为等待连接
  • 蓝色的Isolated为电源正常但是网线掉线，可以使用无线Mesh进行连接
  • 红色的Disconnected为掉线
