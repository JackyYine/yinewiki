page d'accueil

Une fois connectés, les utilisateurs peuvent cliquer sur les boutons de la barre de navigation supérieure pour changer de page :

 

  • CommencerUtilisé pour afficher l'interface du bureau
  • Réserve Utilisé pour afficher les informations de rendez-vous
  • Sortir pour afficher toutes les informations de livraison
  • Commande pour afficher les informations historiques sur les commandes et le résumé quotidien
  • Client pouvez afficher toutes les informations enregistrées sur les clients à emporter (les informations client liées aux factures ne seront pas affichées ici, si vous souhaitez afficher les informations sur l'entreprise cliente ou P.IVA, veuillez vous référer à la colonne suivante)
  • Facture Utilisé pour afficher toutes les informations de facturation et les informations client de facturation Utilisé pour définir divers paramètres internes du logiciel
  • Reglages pour configurer divers paramètres dans lelogiciel
