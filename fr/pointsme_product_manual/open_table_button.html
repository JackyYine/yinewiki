Parallèle

 

Il est utilisé pour fusionner deux tables, par exemple, lorsque 4 personnes sont assises sur deux tables pour deux, vous pouvez utiliser cette fonction pour fusionner des tables pour en faire une grande table. 

Comment ça fonctionne: Sélectionnez deux tables ou plus que vous souhaitez fusionner, puis cliquez sur le bouton Fusionner et confirmez

 

bifurquer

 

Il sert à répartir les personnes et les plats d'une table à une autre table libre. 

Mode d'emploi : sélectionnez la table que vous souhaitez diviser et une table libre cible, puis cliquez sur Diviser et confirmez

 

changer de chaîne

 

Il est utilisé pour basculer tous les plats et informations d'une certaine table vers une autre table.

Mode d'emploi: Sélectionnez une table que vous souhaitez modifier et une table libre cible, puis cliquez sur Modifier et confirmez

 

étendre

 

Utilisé pour prolonger le temps d'occupation d'une table

Mode d'emploi : sélectionnez le tableau dont vous souhaitez prolonger le temps, puis cliquez sur Prolonger, entrez le temps que vous souhaitez prolonger (minutes) et confirmez

 

changer le type de menu

 

Utilisé pour changer le type de menu de la table sélectionnée, comme changer le menu du midi en menu du soir ou en menu à la carte

Remarque : Il ne peut être appliqué qu'aux tables ouvertes et n'ayant pas passé de commande. Si vous souhaitez modifier le type de menu après avoir passé une commande, veuillez consulter le bouton Modifier le nombre de personnes sur la page des détails de la table.

Mode d'emploi : Sélectionnez la table dont vous souhaitez modifier le menu, puis cliquez sur Modifier le type de menu. Après avoir sélectionné à nouveau le type de menu, saisissez le nombre de convives et confirmez.

 

 

 

table temporaire

 

Utilisé pour créer une table temporaire, cette table ne restera pas sur la page d'ouverture en tant que console mais disparaîtra après le paiement et le classement. 

Remarque : Il ne peut être appliqué qu'aux tables ouvertes et n'ayant pas passé de commande. Si vous souhaitez modifier le type de menu après avoir passé une commande, veuillez consulter le bouton Modifier le nombre de personnes sur la page des détails de la table.

Mode d'emploi : Cliquez sur la table temporaire, entrez le numéro de table, sélectionnez le type de menu, entrez le nombre de personnes et confirmez

 

table temporaire

Utilisé pour créer une table temporaire, cette table ne restera pas sur la page d'ouverture en tant que console mais disparaîtra après le paiement et le classement.

Mode d'emploi : Cliquez sur la table temporaire, entrez le numéro de table, sélectionnez le type de menu, entrez le nombre de personnes et confirmez

description du bloc de tableau

 

De haut à gauche en bas à droite : 

Partie supérieure:

En haut à gauche : nombre de convives/nombre maximum de personnes, en haut au milieu : mot de passe de la table, en haut à droite : heure d'ouverture (heure : minute)

Central:

Milieu supérieur : numéro de table Milieu milieu : temps écoulé d'ouverture de la table (minutes) Milieu inférieur gauche (rouge) : état du paiement (prépayé/paiement)

Partie inférieure : 

Milieu inférieur : état de la table : 1. Table vide ; 2. Table ouverte (pas encore passé de commande) ; 3. Montant de la consommation (déjà passé une commande) ; 4. Table de compensation (besoin d'ouvrir l'état de compensation de la table) en bas à gauche (point jaune ): S'il y a un point jaune, cela signifie que le tableau a formé une cotation

L'addition, s'il vous plaît

 

Lorsque le client paie seul (ou ensemble), cliquez sur le bouton de paiement pour former une facture, qui collecte automatiquement tous les plats et dépenses

 

Partager un compte

 

Lorsque les clients veulent payer les plats qu'ils ont commandés, ils cliquent sur le bouton de partage et sélectionnent les plats payés par la première personne et le nombre de places (ou de forfaits repas) payés par la première personne. Après sélection, la première facture sera formé après confirmation. À l'étape suivante, vous pouvez choisir de continuer à sélectionner des plats pour former une deuxième facture ou cliquer sur Annuler pour saisir « Facture 1 » afin de payer la première facture. Conseils : Lorsqu'il ne reste que la dernière personne, vous pouvez annuler et cliquer sur Commander pour ajouter tous les plats et personnes restants à la dernière facture.

 

Par habitant

1. Lorsque le client souhaite diviser la facture par personne, cliquez sur le bouton per capita, entrez le nombre de personnes pour diviser le compte et confirmez, et une facture correspondant au nombre de personnes sera automatiquement générée ; le caissier peut effectuer différentes opérations de caisse correspondant à chaque facture.

 2. Le client souhaite payer un montant fixe (si un client part en premier à une table et souhaite d'abord donner 30 yuans), cliquez sur le bouton per capita, changez le type en "montant", puis entrez le montant correspondant et confirmez . Ensuite, une facture avec le montant correspondant sera formée et l'opération de paiement peut être effectuée en premier. Plus tard, les personnes peuvent choisir de régler la partie restante de la facture en fonction du montant ou de partager la facture par personne.

 

Annuler le fractionnement

Ce bouton apparaîtra lorsqu'il y aura au moins 1 facture impayée ; après avoir cliqué dessus, toutes les factures formées et impayées seront rapidement supprimées.

Pourcentage de remise

Mode de paiement

 

Cette zone permet d'attribuer le mode de paiement utilisé par le client, par défaut c'est tout en espèces. Si le client paie par carte, après avoir cliqué sur la carte, il peut double-cliquer sur la zone de saisie pour renseigner directement le prix final sans saisie manuelle. 

Remarque : Si le client paie de plusieurs manières, le montant en espèces doit être saisi en dernier, par exemple : paiement en espèces + carte de crédit, vous devez d'abord saisir le montant de la carte de crédit, puis saisir le montant en espèces ; sinon, d'autres montants sera remis à zéro

 

Code de loterie

 

Cliquez ici pour entrer le code loterie client (LOTTERIA SCONTRINO) et l'imprimer sur le reçu fiscal. 

Remarque : La saisie du code de loterie nécessite que le type de reçu soit « reçu fiscal » et que le mode de paiement soit « carte magnétique » pour prendre effet.

Les points affichés seront actualisés immédiatement après le paiement (remarque : vous devez d'abord définir les règles de points dans la gestion des clients)

Reçu fiscal

 

Cliquer sur ce bouton imprimera un reçu fiscal (SCONTRINO FISCALE) à partir de l'imprimante fiscale et changera le statut de cette facture en "Payé Fermé".

 

Facture

 

Ce bouton peut être utilisé pour émettre une facture au client. Après avoir cliqué, la fenêtre d'informations sur l'entreprise du client apparaîtra. Après l'avoir remplie, cliquez sur la facture et deux factures seront imprimées à partir de l'imprimante pré-imprimée (une pour le client et un pour vous-même), et le statut de la facture passe à "Fermé". 

Les clients qui ont émis des factures (ou ajoutés manuellement dans "Invoice Library-Buyer Information Management") seront automatiquement enregistrés dans la base de données, et vous pourrez rapidement rechercher des informations via "Business Name" ou "Company Tax ID P.IVA". Une fois la fonction de recherche de numéro fiscal activée, l'icône de recherche à gauche du "numéro fiscal de la société P.IVA" deviendra rouge. 

Après avoir cliqué, vous ne pouvez saisir que le numéro fiscal de la société du client pour obtenir directement le reste des informations requises auprès du bureau des impôts (le code de reçu/boîte aux lettres légale n'est pas Vous l'obtiendrez, vous devez le saisir manuellement), si vous souhaitez activer cette fonction, vous pouvez contacter notre service client pour consultation. 

Remarque 1 : Vous n'avez besoin de remplir qu'un seul des codes de réception et de la boîte aux lettres légale (choisissez 1 parmi 2). Si vous remplissez la boîte aux lettres légale, vous devez réserver 0000000 dans le code de réception, ne le supprimez pas.

Remarque 2 : Numéro d'identification fiscale général CF, EMAIL, numéro de téléphone portable ne sont pas requis 

Remarque 3 :  Après l'émission de la facture, la facture n'a pas été automatiquement livrée au bureau des impôts. Vous devez vous rendre dans la bibliothèque des factures et cliquer sur livrer. Vous pouvez l'utiliser immédiatement ou une fois par jour, ou vous pouvez accéder aux paramètres du système pour activer la livraison automatique à intervalles réguliers.

 

Payer l'addition

Archive

 

Lorsque l'une des opérations ou actions de paiement ci-dessus est effectuée, le système vérifie l'état de la table ; si tous les frais sont réglés et que l'état de tous les plats est "servi", la table sera automatiquement fermée et vidée. 

Si elle n'est pas fermée automatiquement, vous pourrez fermer et vider la table plus tard en appuyant sur le bouton archiver (les plats non servis seront automatiquement remboursés, et le montant restera inchangé)

 L'archivage ne pourra être effectué qu'après règlement

Descriptif du champ

 

état de la caisse

 

Le statut de paiement de la facture en cours est initialement "non clôturé", et il deviendra "clôturé" après l'une des trois opérations de caisse (opérations de caisse : "reçu fiscal", "facture", "paiement")

 

Type de note

 

Comment la facture actuelle est réglée. Initialement, c'est "Aucun", et après l'opération de paiement, il changera selon l'opération : "Tax Receipt" (reçu fiscal), "Invoice" (facture), "Invoicing" (facture de paiement).

 

Montant de la remise

 

La différence entre la facture actuelle et le montant initial. La valeur initiale est 0, et elle changera en conséquence lorsqu'une opération de remise est effectuée (définir un pourcentage de remise ou modifier directement le prix final)

 

Monnaie en espèces

 

La différence entre le montant du paiement (espèces, carte de crédit, ticket restaurant, etc.) et le prix final permet de calculer rapidement la monnaie à rendre au client.

Description du bouton

 

Afficher les articles commandés

 

Après avoir cliqué, vous pouvez voir les plats qui ont été commandés à cette table (les plats qui ont été commandés seront marqués en bleu), et cliquer à nouveau pour les masquer à nouveau.

 

Ajouter des plats

 

Vous pouvez accéder rapidement à la gestion des menus dans les paramètres

 

Commander par code plat

 

Vous pouvez entrer le code plat pour commander rapidement des plats : utilisez le clavier pour saisir le numéro de plat correspondant, puis appuyez sur Entrée pour l'ajouter directement. Si vous souhaitez commander plusieurs exemplaires, vous pouvez directement appuyer plusieurs fois sur Entrée pour ajouter le nombre d'exemplaires. Lorsque vous avez terminé, cliquez sur Terminer pour fermer la fenêtre.

 

Bascule de menu

 

Cliquez sur le bouton dans le coin supérieur droit (nommé le nom du menu actuel) pour changer rapidement de menu et commander des plats à partir d'autres menus. Exemple : Un client qui mange un repas à forfait veut commander un seul plat, vous pouvez changer de menu et commander un plat avec un prix

 

Colonne de type plat

 

L'extrême droite de l'interface de commande est la colonne du type de plat, vous pouvez cliquer sur un type pour afficher ses plats

 

Ne pas imprimer

 

Après avoir coché Pas d'impression, le plat n'entrera pas dans la cuisine et ne sera pas imprimé après la commande. Vous pouvez rapidement sélectionner ou désélectionner tous les plats grâce à la case à cocher en haut sans imprimer.

 

Quantité

 

Dans le panier, vous pouvez rapidement ajouter ou supprimer chaque plat ajouté au panier. Si vous souhaitez supprimer un plat du panier, vous devez cliquer sur la croix (X) à droite.

 

Remarque

 

Lorsque vous cliquez sur le bouton "Remarques" d'un plat dans le panier, vous pouvez ajouter et modifier ses remarques et supprimer des matières premières. La saisie dans la case des remarques sous le panier ajoutera des remarques correspondantes à tous les plats du panier lors de la commande (lors de l'utilisation d'une imprimante, elles seront affichées dans les "remarques complètes de la commande" ci-dessus).
