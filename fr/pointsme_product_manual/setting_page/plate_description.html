Description du champ Plats

Les champs avec un astérisque rouge sont obligatoires.

Code de plat

Code de plat ; il est obligatoire et ne peut pas être répété.

Nom du plat

Le nom du plat ; le nom peut être saisi dans différentes langues via le bouton de traduction à côté. La langue des plats affichés sur le terminal tablette (serveur, client), le terminal mobile du serveur, et le terminal desktop (caissier, serveur) est la langue du logiciel de commande ; la langue du terminal WEB est la langue du système du téléphone portable (ou de l'ordinateur). Si la langue correspondante n'est pas définie, le nom par défaut sera obtenu. Le côté cuisine affichera le nom par défaut (ou alias : voir ci-dessous).

Plats

Le surnom du plat ; après avoir renseigné, vous pouvez définir le nom d'affichage/impression de l'arrière cuisine et/ou du point de restauration (vous pouvez cocher « affichage centre de production » et « affichage point de vente légumes » comme « alias plat »). Lorsque le mode d'impression est activé, vous pouvez définir le nom d'impression, l'alias ou les deux dans les paramètres système - les paramètres de l'imprimante en un seul clic.

Lettre initiale

Après avoir rempli, vous pouvez rapidement rechercher des plats par les premières lettres sur la page du menu.

Code à barre

Après avoir rempli, vous pouvez le définir et l'imprimer dans le contenu de l'étiquette de prix.

Prix

Le prix unitaire par défaut du plat (si le prix est différent dans différents menus, vous pouvez modifier le prix correspondant dans la liste des menus ultérieurement) Lorsqu'il est défini sur le prix réel, une case apparaîtra automatiquement lors du paiement et le caissier entrez le prix correspondant.

Type de plat

Sélectionnez le type de plat pour ce plat.

Priorité

La priorité des plats ; lorsque le centre de production utilise l'écran et est une ancienne version de la cuisine, les plats non préparés trieront les plats avec une priorité supérieure à l'avant (les plats avec le même temps de commande).

Commande de repas

L'ordre de repas par défaut des plats ; il sera utilisé uniquement lorsque le système est réglé sur l'ordre de repas automatique (paramètres du système-réglage de la commande-démarrer l'ordre de repas automatique). L'opération spécifique peut être vue dans l'ordre de restauration

Le nombre maximum de plats combinés

Le nombre maximum de fusion automatique de cette parabole dans le centre de production n'est valable que pour le centre de production utilisant le moniteur.

Ancienne version de la cuisine : le même plat qui n'est pas en cours de préparation sera fusionné dans le même bloc jusqu'à ce que le nombre maximum de plats fusionnés soit atteint, puis il sera affiché séparément et transféré dans un autre bloc pour un nouveau cycle de fusion.
Remarque : Si le nombre de commandes uniques passées sur une table dépasse le nombre combiné maximum, il sera toujours affiché dans une boîte et les copies excédentaires ne seront pas séparées.

Nouvelle version de la cuisine : en cliquant sur le nombre de plats fusionnés à côté du plat, le nombre de plats qui sont automatiquement sélectionnés pour servir est le nombre maximum de plats fusionnés par défaut. Si vous souhaitez ajouter plus de plats en un clic, vous besoin de vérifier manuellement plus de tables ou de faire glisser la bande de portion

En stock

La quantité d'inventaire du plat : Après le réglage, chaque fois que le plat est commandé, le nombre de portions correspondant sera déduit de l'inventaire. Les quantités en stock peuvent être vues lors de la prévisualisation du menu. Lorsque l'inventaire atteint 0, la parabole affichera le champ "épuisé" et ne pourra pas être ajoutée au panier. Lorsqu'un plat est en rupture de stock, mais que vous souhaitez que les clients voient que le plat est disponible dans le restaurant lors de la prévisualisation du menu, mais qu'il est actuellement en rupture de stock, vous pouvez utiliser cette fonction pour régler directement le stock à 0. Cela ne pas besoin d'être supprimé des plats du menu.

Image

Cliquez pour télécharger l'image du plat, et le client pourra la voir sur le menu après le téléchargement.

Matières premières personnalisées

Dans les plats normaux, cette option est "Non". Réglez sur "Oui" lorsqu'il est nécessaire de sélectionner divers ingrédients de matières premières (tels que POKE).

Matière première

Les ingrédients de ce plat. Cliquez sur la bibliothèque de matières premières et vérifiez de quelles matières premières ce plat est fait. Après le réglage, les clients peuvent accéder à la page des détails du plat pour afficher ces informations lors de la commande.

Des matières premières peuvent être ajoutées

Cliquez sur la bibliothèque de matières premières et vérifiez quelles matières premières supplémentaires peuvent être ajoutées à ce plat. Après le réglage, les clients peuvent accéder à la page des détails du plat pour afficher ces informations lors de la commande et choisir rapidement d'ajouter des matières premières.

La matière première peut être supprimée

Après avoir cliqué sur la bibliothèque de matières premières, la liste des matières premières de ce plat s'ouvrira. Après avoir sélectionné la matière première correspondante, vous pouvez supprimer rapidement une matière première sur la page de détail du plat.

Méthode de production

Une description du plat. Après avoir rempli, les clients peuvent accéder à la page des détails du plat pour afficher ces informations lors de la commande.

Préférence gustative

Cliquez sur Afficher pour ouvrir la bibliothèque des préférences de goût. Après vérification, les clients et les serveurs peuvent vérifier rapidement une préférence de goût sur la page des détails du plat (généralement une note commune sera définie, telle que pas de glace)

Contient des allergènes

Cliquez sur Afficher pour ouvrir la bibliothèque d'allergènes. Après vérification, les clients et les serveurs peuvent voir ces informations sur la page des détails du plat et peuvent utiliser la fonction de dépistage des allergènes du menu pour bloquer les plats contenant des allergènes filtrés.

Centre de production

Quel centre de production a fabriqué ce plat

Affichage du centre de production

Quel nom est affiché pour le centre de production (cuisine)

Affichage du distributeur

Quel nom est affiché dans le menu

Le compte client ne prend pas en charge le type de client

Ce plat peut être bloqué d'un bout à l'autre du client (non affiché sur la carte).

Produit semi-fini

Ce plat nécessite-t-il plus d'une personne pour le préparer

Prend en charge l'impression rapide

Réglez sur "Oui" pour sélectionner et imprimer dans la fonction "Reçu rapide"

Étiqueter

Affichez l'étiquette correspondante sur le menu (coin supérieur gauche de l'image).

Ajouter à la remise totale sur la facture

Lorsqu'il est défini sur "Non", ce plat ne sera pas pris en compte lors de l'octroi d'un pourcentage de réduction sur la facture

Catégorie

La catégorie de mets (plats, boissons, vins) permet d'afficher différents taux de taxes.
