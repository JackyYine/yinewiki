Définir l'interface

 

 

Informations d'affaires

Ici, vous pouvez afficher les informations sur votre propre magasin, les informations sur l'entreprise, les coordonnées de l'adresse ; afficher/télécharger le LOGO du magasin, modifier l'IP du réseau local et public (pour les codes QR avec tableaux) et définir le mot de passe du nom WIFI (pour l'application client)

 

 

 

Centre de compte

Gestion de compte

Ici, vous pouvez créer des comptes pour chaque extrémité à utiliser et gérer les comptes (réinitialiser les mots de passe, supprimer, désactiver, modifier les autorisations, etc.)

Étapes pour ajouter un compte :

1. Cliquez sur Ajouter un compte

2. Entrez le compte (nom d'utilisateur de connexion), le mot de passe (mot de passe de connexion), le nom de l'employé, le numéro de travail et sélectionnez le type de compte et le rôle

3. Réglez le statut sur "Activé" et confirmez

Si vous souhaitez désactiver temporairement un compte, vous pouvez changer le statut en "Désactivé", et si vous souhaitez supprimer définitivement un compte, utilisez la fonction de suppression.

 

Type de compte

Administrateur : uniquement applicable au bureau ; pour les patrons, les actionnaires, les gérants de magasin, etc.

Caissier : applicable uniquement sur ordinateur ; utilisé par les gérants de magasin, les caissiers, etc.

Salle de course : Applicable au terminal mobile de la salle de course (Android, iOS), au terminal de la tablette (Android, iOS) et au bureau ; pour la salle de course ou tout initié qui souhaite commander avec un téléphone mobile (patron, gérant de magasin, etc.), le terminal de bureau est destiné à l'ordinateur du restaurant pour ajouter des commandes

Client : applicable au terminal de tablette (Android, iOS ); les modes suivants peuvent être définis :

 

Table non liée (compte client général) : vous pouvez ouvrir n'importe quelle table (la table qui a déjà été ouverte doit entrer le mot de passe de la table)

 

Table de liaison (compte client individuel) : après vous être connecté, vous ne pouvez voir que la table liée et la saisir, pas besoin de saisir le mot de passe de la table

 

Clients à emporter : une fois connectés, ils ne peuvent voir que le menu du colis et passer une commande de colis (les clients peuvent entrer leur numéro de téléphone portable, leur nom et d'autres informations)

 

 

Arrière-cuisine : Applicable au terminal de tablette (Android uniquement) et au terminal de cuisine Windows ; besoin de lier le centre de production correspondant

Port de ramassage des aliments : applicable au terminal de tablette (Android uniquement) et au terminal de cuisine Windows ; besoin de lier le port de ramassage des aliments correspondant

Utilisateurs publicitaires : applicable aux appareils Android (tablettes, grands écrans de boîtiers Android, etc.) ; après la connexion, les publicités définies en arrière-plan peuvent être lues

Écran caissier secondaire : adapté aux tablettes (Android uniquement) ; il sert à afficher le code QR de la loterie lors du paiement des clients ; il doit lier le compte habituellement utilisé pour le caissier

 

 

 

Gestion des rôles

Ici, vous pouvez ajouter des rôles pour attribuer différentes pages, et les autorisations des boutons sont données à chaque compte

Le système est livré avec 4 rôles par défaut :

 

Titulaire : possède toutes les autorisations

 

Gérant de magasin : Identique au titulaire

 

Caissier : aucune autorisation pour définir et afficher la liste des commandes

 

Salle d'attente : pas d'autorisation d'installation, de consultation des commandes et de diverses opérations de paiement

liste des menus

Ici, vous pouvez visualiser, modifier, supprimer les informations du menu, les plats du menu, etc.

 

Menu Ajouter/Modifier la description du champ

Cliquez sur Ajouter un menu pour ajouter un nouveau menu, et cliquez sur Modifier pour modifier les informations du menu correspondant.

 

Nom du menu

Le nom par défaut du menu ; vous pouvez ajouter la traduction correspondant à chaque langue grâce au signe plus à droite

 

Que ce soit à temps plein

Lorsqu'il est réglé sur "Oui", le client affichera ce menu à tout moment (permet aux clients de choisir quand le réglage du menu est activé, sinon seul le menu avec la priorité la plus élevée sera affiché)

Lorsqu'il est réglé sur "Non", le réglage de l'heure du repas apparaîtra ci-dessous, et il ne sera affiché sur le client que pendant la période de temps définie. Différents prix et frais de siège peuvent être fixés pour différentes périodes.

 

Que ce soit pour activer

S'il n'est pas activé, ce menu sera bloqué de toutes parts jusqu'à ce qu'il soit modifié pour être activé.

 

Types de menus

Commande : Ajoutez des plats à ce type de menu, le prix par défaut est le prix du plat en arrière-plan ; vous pouvez définir le prix de la place.

 

Forfait repas : Ajoutez des plats à ce type de menu, le prix par défaut est 0 ; vous pouvez définir le prix du siège, le prix du repas pour les adultes, le prix du repas pour les enfants, le nombre maximum de plats par personne et le temps d'intervalle lors de la commande.

 

Forfait : ajoutez des plats à ce type de menu, le prix par défaut est le prix des plats en arrière-plan ; vous pouvez définir si vous souhaitez l'activer sur la plateforme.

 

Intervalle de commande/Nombre maximum de plats

Ce paramètre n'est disponible que pour les menus des forfaits repas

Nombre maximum de plats : Contrôlez le nombre maximum de plats (plats avec un prix de 0) que les utilisateurs (par personne) peuvent commander en une seule commande pour éviter les commandes malveillantes. Ne pas remplir ou 0 signifie illimité.

Intervalle de commande : contrôlez le temps entre deux commandes passées par l'utilisateur pour éviter les commandes fréquentes. Ne pas remplir ou remplir 0 signifie aucune limite.

Les deux options doivent être configurées pour être utilisées. Si le nombre maximum de plats est fixé à 7 et que le temps d'intervalle est de 10 minutes, une table pour 2 personnes peut commander 14 plats avec un prix de 0, puis 14 plats peuvent être commandés à nouveau après 10 minutes.

 

Tarif adulte/tarif enfant/tarif siège

Après réglage, après saisie du nombre de personnes lors de l'ouverture d'une table, le tarif repas adulte/repas enfant/siège sera automatiquement calculé.

Remarque : Si un fuseau horaire est défini, le calcul des frais dans le fuseau horaire sera utilisé pour ouvrir la table dans le fuseau horaire ; le frais par défaut sera utilisé pour calculer l'ouverture de la table en dehors du fuseau horaire (serveur ou caisse enregistreuse).

 

Si la plateforme est activée

Seuls les plats de menu de type forfait auront des paramètres secondaires

Lorsqu'il est défini sur "Oui", ce menu peut être utilisé sur la plateforme de vente à emporter (page Web, application, lecture de code QR à emporter).
