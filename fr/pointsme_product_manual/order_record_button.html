Page principale de la liste de commandes

Les comptes autorisés peuvent afficher toutes les informations historiques sur les commandes sur cette page, et peuvent rapidement afficher et exporter les commandes qu'ils souhaitent trouver via des conditions de filtrage.

 

Description des boutons de fonctionnement

Résumé quotidien

Cliquez pour confirmer et imprimer le résumé quotidien de l'imprimante fiscale (CHIUSURA GIORNALIERA)

 

Exporter

Exporter les informations de la liste de commandes des résultats de la recherche (fichier EXCEL)

 

Liste de factures

Cliquez pour entrer dans la liste des factures

 

Vérifier les détails

Cliquez pour accéder à la page de détails de la commande correspondante. Sur cette page, vous pouvez vérifier les informations sur le plat de la commande, les informations de paiement, réimprimer la préimpression, la facture, le reçu fiscal, entrer le code de loterie, afficher le journal des opérations, etc.

 

Page Liste des factures

Vous pouvez afficher toutes les informations historiques sur les factures sur cette page, et vous pouvez afficher rapidement les factures que vous souhaitez trouver grâce aux conditions de filtrage.

Quelle est la différence entre une facture et une commande ? Une commande peut avoir plusieurs factures

 

Imprimer

Après avoir cliqué, vous pouvez imprimer un tableau de données d'entreprise rapide (selon les conditions de filtre)

 

Vérifier les détails

Cliquez pour accéder à la page des détails de la facture. Sur cette page, vous pouvez afficher les informations de menu, les informations de paiement, la préimpression de réimpression, le reçu fiscal, la facture, saisir le code de loterie, etc.

 

Retour

Cliquez pour revenir à la page de la liste des commandes
