Page principale de la liste des clients

Cette page peut afficher et gérer les informations client, telles que le nom, les coordonnées, l'e-mail, l'adresse, la date de naissance, les points, le montant de la consommation cumulée, etc.

Remarque : Si vous souhaitez activer la fonction de points client, vous devez configurer les règles de points dans les paramètres.

Les informations client peuvent être injectées des manières suivantes :

1. Saisir manuellement à partir du bouton "Ajouter" sur cette page ;

2. Entrez les informations du nouveau client lors de l'ajout d'une commande à emporter ;

3. Sur la page de paiement, cliquez sur "Ajouter un nouveau client" pour entrer ;

4. L'utilisateur s'enregistre lui-même à partir de l'APP.

 

Description des boutons de fonctionnement

modification des points

Vous pouvez ajouter ou soustraire manuellement des points aux clients correspondants (par exemple, quelles récompenses le client a échangées avec les points, vous pouvez déduire d'ici et écrire une note)

 

Historique des points

Vous pouvez visualiser tous les crédits du client correspondant (méthode d'acquisition, temps d'acquisition, etc.)

 

Modifier

Vous pouvez modifier les informations client correspondantes (nom du client, numéro de téléphone portable, adresse, email, anniversaire)

 

Supprimer

Supprimer toutes les informations correspondant au client
