Rendez-Vous

Les rendez-vous peuvent être gérés sur cette page (voir, ajouter, modifier)

Grâce à la boîte de critères de recherche ci-dessus, vous pouvez effectuer une recherche rapide

 

Ajouter un rendez-vous

1. Cliquez sur "Ajouter un rendez-vous" 2. Sélectionnez la date et l'heure, saisissez le nombre de personnes et cliquez sur Requête 3. Sélectionnez le tableau libre qui apparaît dans le système 4. Saisissez les informations du client (numéro de téléphone portable, nom) et les remarques, et cliquez sur OK

 

Modifier rendez-vous

Si vous souhaitez modifier les informations de réservation, vous pouvez cliquer sur "Modifier" dans l'option de réservation correspondante. Si vous souhaitez annuler le rendez-vous, cliquez sur "Annuler"

 

Réserver une table

Lorsque le client arrive au restaurant, cliquez sur le bouton "ouvrir la table" à côté de la réservation correspondante, sélectionnez le type de menu, saisissez le nombre de convives et validez.
