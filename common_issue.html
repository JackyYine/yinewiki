打印机刀口卡住导致纸盖打不开

打印机运作时有小概率出现刀口卡住纸盒导致无法出纸，这种情况下需要按照以下方式操作：

  1. 关闭打印机电源
  2. 打开打印机带有锯齿的前盖
  3. 用手或者任意工具拨动齿轮直到刀片完全收缩回打印机内(根据打印机型号拨动方向会有所不同，GP-C80250i plus是朝向出纸口方向拨动)
  4. 将前盖装回打印机
  5. 开机测试是否恢复正常

扫码点餐扫不出来或失效

检查以下几点: 

  1. 检查餐厅外网是否有网络
  2. 检查外网IP是否改变，使用ddns的情况下检查IP更新是否正常。
  3. 检查内网与外网路由器的端口映射设置
  4. 检查软件内 设置–商家信息–局域网IP 是否符合当前外网IP

热敏打印机不出纸

检查以下几点:

  1. 打印机是否开机，是否有纸
  2. 检查打印机背后网线是否亮起，试图ping打印机IP，查看是否有回应
  3. 检查软件内打印机是否设置正确
  4. 检查Windows打印机设置是否正确

通常大多数情况都为网线连接不良导致无法与主机通信从而不出纸。 个别情况为设备硬件损坏，这种情况需要联系客服保修。

如何设置扫码点餐

设置扫码点餐分为以下步骤:

  1. 确保公网IP为固定IP或者已设置 DDNS
  2. 在路由器内设置端口映射 教程
  3. 将桌面端软件内的 设置–商家信息–局域网IP 设置为当前的公网IP或DDNS地址
  4. 进入桌面端软件内的 设置–桌台管理–桌台管理，点击右上角的批量打印二维码即可获得包含所有桌台二维码图片的压缩包
  5. 选择任意二维码进行扫码测试是否正常。

如何制作克隆硬盘

通过克隆（Clone）的方式可以快速复制一个硬盘内的所有文件至另一块硬盘，从而达到量产初始化完成的硬盘或备份已有硬盘的目的。

制作方式

  1. 在Macrium官网下载备份软件 Macrium Reflect free 并安装在电脑上。
  2. 在电脑上插上旧盘与新盘 （如果是想将当前电脑的数据备份的话只需插入新盘即可）。
  3. 打开Macrium Reflect点击旧盘后点击克隆此硬盘 图1
  4. 选择目标盘，之后一直点击下一步直到克隆进程开始。
  5. 等待克隆结束点击完成即可。

平板登录不了如何解决

优先检查以下几点

  • 检查账号与密码是否正确
  • 检查WiFi是否连接至名称为PointsMe的WiFi。（某些商家可能没有使用我们的WiFi设置，这种情况下需要连接进与主机同一网段的WiFi。）
  • 检查主机防火墙设置，查看是否允许16961端口进站与出站的通信
  • 检查平板APP版本是否与主机符合
  • 重新创建一个新账号，查看是否能登录

在以上几点都检查过并且确认正常后还是无法登录的情况下联系客服。

Windows打印服务异常

症状
  • 打印延迟极其高
  • 打印服务占用大量内存（RAM）
  • 打印服务占用大量磁盘读写百分比
解决方法
  1. 关闭打印服务（Print spooler），可通过服务管理器或者直接在拥有管理员权限的CMD内输入此指令 net stop spooler
  2. 在开始的搜索栏输入 %WINDIR%\system32\spool\printers，删除打开的文件夹内的所有文件
  3. 重启电脑，并确认重启后打印服务恢复正常
  4. 若打印服务还是处于关闭状态则需要手动打开打印服务，可通过服务管理器或者直接在拥有管理员权限的CMD内输入此指令 net start spooler

单个桌台账号登录多个平板导致重复出菜

正确操作为将多张桌合并为同一张，并且给每台平板登录不同的账号。 例: 15号桌是永久大桌，那最好准备15a与15b两个桌台作为需要多个平板时合并用的桌台，然后三个平板分别登录15，15a 与 15b 三个不同的账号。 

  • 例: 4，5，6三张桌合并为临时大桌，那三个平板可以分别登录4,5,6三个账户。
  • 错误例子: 15号桌为大桌，此时需要三个平板，在三台平板上皆登录15号桌的账号。

平板图片刷新缓慢

图片的加载速度跟网速和图片大小有关，倘若客户使用的是平板，则只需要检查图片大小是否过大，压缩图片即可解决。

若是手机扫码出现加载过慢，除了图片大小以外还需使用 Speedtest 检查客户上载速度，如果在20Mbit/s以下则建议客户升级网络套餐，提高上载速度（Upload）。
